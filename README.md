## Project Description
The main purpose of this project is to build a machine learning model that accurately predicts the sales of different product families at Favorita stores. On the business side, this helps to:
      
    1. Decrease the possibility of food waste related to overstocking.
    2. Ensure that retailers have just enough of the right products at the right time.
    3. Improve customer satisfaction.
